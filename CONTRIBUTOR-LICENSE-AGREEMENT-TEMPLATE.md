# Contributor License Agreement Template

This is a project template, not legal advice. Sabinok Corporation should have
counsel review the final agreement before accepting outside contributions.

## Required Terms

For substantive pull requests, Sabinok Corporation should have a signed
contributor license agreement on file that does at least the following:

- identifies the contributor and, if applicable, the contributor's employer
- confirms that the contributor has the right to contribute the work
- states that, in consideration for the right to contribute to the project, the
  contributor grants Sabinok Corporation an irrevocable, worldwide,
  royalty-free, non-exclusive, sublicensable right to use, copy, modify,
  distribute, publicly perform, publicly display, and relicense the contribution
- confirms that the contribution may be released publicly under
  `GPL-3.0-or-later`
- confirms that the contribution does not intentionally include third-party code
  or data under incompatible terms
- includes the contributor's signature and date

## Suggested Pull Request Note

After the agreement is signed and stored outside the public repository, the
maintainer should add a private note or label to the pull request:

```text
Contributor license agreement on file for <contributor name>; verified by <maintainer>
on <date>.
```
