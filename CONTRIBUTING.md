# Contributing

Sabinok PDF is a Sabinok Corporation project. The public repository is licensed
under `GPL-3.0-or-later`, and Sabinok Corporation may also offer the project
under less restrictive licenses.

Because dual licensing requires clean rights, do not merge substantive outside
pull requests unless Sabinok Corporation has a signed contributor license
agreement on file for the contributor. Contributors keep their copyright; the
agreement grants Sabinok Corporation the durable right to relicense the
contribution.

## Pull Request Policy

Before a substantive outside contribution is merged:

- the contributor must sign the contributor license agreement in
  [CONTRIBUTOR-LICENSE-AGREEMENT-TEMPLATE.md](CONTRIBUTOR-LICENSE-AGREEMENT-TEMPLATE.md),
  or a counsel-approved replacement
- a Sabinok maintainer must confirm that the signed agreement is on file outside
  the public repository
- signed agreements must be emailed to legal@sabinok.com, not submitted by pull
  request
- the pull request must not include third-party code, generated output, private
  PDFs, credentials, or data copied from incompatible sources
- the contribution must be licensed to the public under `GPL-3.0-or-later`

Trivial typo fixes and issue comments generally do not need an agreement, but
when in doubt, require the agreement before merging.

Do not rely on a Developer Certificate of Origin alone for this repository. A
DCO is useful provenance evidence, but it does not by itself give Sabinok
Corporation the broader relicensing rights needed for future non-GPL licensing.
